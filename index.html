A Tiny Bit Of Code For Dummys

C

 Step 1. Add this thing at the top of your text editor called a header a thing to basically tell your machine "Require this Package" a thing that adds different functionallitys to your code. Rememeber to add #include before st.dio the basic input/output Package within the <> signs.

Step 2. Add int main() after the next 2 lines, then add the opening and closing {} after.

Step 3. To print a line of text type the printf function as shown: printf("Hello, world!\n"); Remember to end it in ;

Step 4. at the bottom just before the closing } add return 0; to tell the machine "Yep this worked!".

The result: your code should look like this:

#include <stdio.h>

int main() {
printf("Hello, world!\n");
return 0;
};


And should print "Hello, world!" to the console if done correctly.
